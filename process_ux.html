    UX Web Design Project


    This was a seven-week-long User Experience (UX) project to create a website for a hypothetical AIGA Design Educatorâ€™s conference in San Luis Obispo, California. I was part of a team of four designers. Together we conducted UX activities such as affinity mapping, the creation of personas, scenarios, empathy maps, and an in-depth information architecture. We also created a fully-functional desktop prototype and used it to conduct guerrilla user testing. We used information gathered through these activities to begin coding a cross-platform website using HTML, CSS, jQuery and JavaScript. We then presented the website in front of a panel of three industry professionals. Below is a link to our website along with process photos and a copy of our process book.
